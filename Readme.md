Para instalar os templates do Identity Server 4:

```
dotnet new -i IdentityServer4.Templates
```

Para criar um projeto Identity Server 4:

1. Entre com Developer Command Prompt for VS 2017

2. Entre na pasta \ASPNETCore2-Parte3\Item01

3. Crie um novo diretório `CasaDoCodigo.Identity`:

```
md CasaDoCodigo.Identity
```

4. Entre no diretório `CasaDoCodigo.Identity`

```
cd CasaDoCodigo.Identity
```

5. Instale o template ****IdentityServer4 with ASP.NET Core Identity**

```
dotnet new is4aspid
```

6. Adicione o projeto criado à pasta `Item01` da solução
